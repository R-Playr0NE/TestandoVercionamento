🚀 Guia Básico de Git

Página simples para testar alterações e versionamento.

1. Configurar Git

git config user.name "SeuUserName"
git config user.email "seu@email.com"

2. Enviar projeto para GitHub

cd pasta-do-projeto

git init
git add .
git commit -m "Primeiro commit"

git branch -M main

git remote add origin URL_DO_REPO

git push -u origin main

3. Atualizar projeto

git add .
git commit -m "minha alteração"
git push

4. Clonar projeto

git clone URL_DO_REPO

cd nome-do-projeto

git pull

5. Versionamento (Branch)

Criar nova versão

git checkout -b nova-versao

Enviar nova versão

git add .
git commit -m "nova versão"
git push origin nova-versao

Voltar para main

git checkout main

Juntar versões

git merge nova-versao
git push
